Eloquent ORM in Laravel

Models

Create a model using Artisan:

php artisan make:model User

Define relationships and attributes:

// User.php
class User extends Model {
    protected $fillable = ['name', 'email', 'password'];

    public function posts() {
        return $this->hasMany(Post::class);
    }
}

Queries

Perform queries using Eloquent:

// Retrieve all users
$users = User::all();

// Retrieve user by ID
$user = User::find($id);

// Create a new user
$user = User::create(['name' => 'John Doe', 'email' => 'john@example.com', 'password' => bcrypt('password')]);

// Update user
$user->update(['name' => 'Jane Doe']);

// Delete user
$user->delete();
